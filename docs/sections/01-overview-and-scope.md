# 1. Overview & Scope

## 1.1 Purpose

The St. Clare College Filing System (SCCFS) is a standalone Windows desktop application that provides institutional document management.

## 1.2 Scope

### In-Scope:
- Secure local authentication
- File ingestion and retrieval
- Shelf-based virtual organization
- AES-256-GCM encryption
- SQLite3 backup

### Out-of-Scope:
- Network sync/cloud backup
- Multi-device access
- Real-time collaboration
- macOS/Linux packaging

## 1.3 Guiding Constraints

| Constraint | Value |
|-----------|-------|
| Target OS | Windows 10 22H2+, Windows 11 |
| Electron | 32.x |
| Node.js | 20 LTS |
| Database | SQLite3 |
